BlackLight

This application is a native app for android which is used to send messages while in low lighting environments such as movie theatres or classrooms.

This app has full e-mail and SMS functionality, including sending and an inbox for both. The application also has a contact browser for both e-mail addresses and phone numbers.

It also includes a settings screen which allows the user to globally change things such as:

  • Screen Brightness
  • Font Colour
  • Font Size
  • Typeface
  • E-mail signature

Wheatley

Wheatley is a web-based application which is then converted to run on mobile devices.

Wheatley is primarily built for the Blackberry PlayBook. It is a sign-in application to be used by companies and offices to keep track of guests.

When a user signs in, it prints a badge for them which has all of their information, a picture of them (taken upon signin), and a QR code. The QR code can be signed at later times to sign in and out without needing to register again.

The application also sends out an e-mail to the person which the user is there to visit. Users which sign in are stored in a database and administrators can view all people that are currently signed in, or have previously visited. There is a search function that can be used to find all times a specific person has ever signed in or out

This application is running on Backbone.js with a heavy dependency on Require.js and is purely based on the MVC structure type. It uses many libraries and plugins, such as jQuery, jQuery Mobile, and Simplemodal.
