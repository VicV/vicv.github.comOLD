League of Legends Companion App

This application is a native Android application that is also built for the Blackberry PlayBook. It is my first ever personal app. It is a companion application fora popular free game. It has a list of every character in the game, and a lot of information about each character. It also includes external links to guide sites, so users can get information on their character and how to play them while they are playing the game.

This application was programmed using Java and all information is gathered and displayed using SQLite databases. The app also includes the ability to import and export specific data tables. It was created to be submitted to the Blackberry AppWorld, and has over 3000 downloads with a 4.5 star rating
